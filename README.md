## Nem Music Player

![Nem](screenshot.png?raw=true "Nem")

* Super sleek interface
* Keyboard control
* Distraction-free
* Notifications

### Controls
**?** -> Show help

**Space** -> Play/Pause

**N or Ctrl/Cmd + Right** -> Next track

**P or Ctrl/Cmd + Left** -> Prev track

**L** -> Like track

**Ctrl/Cmd + F** -> Search

## Development environment

Tested on Ubuntu and Elementary OS.

#### Install dependencies 
>npm install

#### Start it with :
>npm start
