## Features

![Nem](screenshot.png?raw=true "Nem")

* Super sleek interface
* Keyboard control
* Distraction-free
* Notification when a track start playing

### Controls
**?** -> Show help

**Space** -> Play/Pause

**N** -> Next track

**P** -> Prev track

**L** -> Like track (coming soon)

## Development environment

Tested on Ubuntu and Elementary OS.

#### Install dependencies 
>npm install

#### Start it with :
>npm start
